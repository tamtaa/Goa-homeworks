1. რა არის display?

"display" განსაზღვრავს როგორ გამოჩნდება ელემენტი: block, inline, flex, grid და სხვ.

2. block და inline ახსნა

Block → ახალ ხაზზე გადადის. Inline → მხოლოდ ტექსტის ზომას იკავებს.

3. block და inline შორის განსხვავება

  • Block იწყებს ახალ ხაზს, Inline — არა
  • Block-ს შეიძლება ჰქონდეს width/height, Inline-ს — არა
  • Block იკავებს მთელ ხაზს, Inline — მხოლოდ შიგთავსის ადგილს

4. 5 მაგალითი display:block-ზე

DIV ყოველთვის block-ია

Paragraph (P) ყოველთვის block-ია

HEADER block-ია
SECTION block-ია
FOOTER block-ია

5.მაგალითი display:inline-ზე

A (ბმული) inline-ია STRONG inline-ია
